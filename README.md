Cycles, Not Trends: A Time Series Analysis of Canadian Lynx Population DynamicsOverviewThis repository contains an applied time series econometrics analysis of the canonical Canadian Lynx dataset (lynx), which records the annual number of Canadian lynx trapped from 1821 to 1934.The project applies the complete Box-Jenkins methodology to demonstrate that the data is best characterized as a cyclical, trend-free, mean-reverting process driven by predator-prey dynamics, rather than a linear or randomly drifting trend.Repository Contentslynx_analysis.R: The core R script containing the full statistical pipeline. It generates all models, performs diagnostic testing, and automatically exports the required visualizations as PDFs.paper.tex: The LaTeX manuscript detailing the theoretical background, methodology, and interpretation of the statistical results.Generated Plots (.pdf): Running the R script will automatically generate several diagnostic and forecasting plots (e.g., fig_raw_series.pdf, fig_arma21_forecast.pdf) necessary to compile the LaTeX document.Methodology PipelineThe analysis is broken down into the following structured steps:Exploratory Data Analysis (EDA): Initial inspection of the 114-year dataset highlighting the extreme variance and dominant $\sim$10-year cycle.Trend & Decomposition: Application of a 10-year centered moving average to confirm the absence of a long-run linear trend.Holt-Winters Smoothing: Fitted as a comparative benchmark to demonstrate the limitations of exponential smoothing on strictly cyclical data.OLS & Feasible GLS Regression: Formal hypothesis testing for a linear time trend. A Generalized Least Squares (GLS) model with an AR(1) error structure is used to correct for the severe residual autocorrelation present in naive OLS.Stationary Transformations: Application of a log transformation (to stabilize variance) and first-differencing (to remove integration and stabilize the mean). Confirmed via Augmented Dickey-Fuller (ADF) testing.ARMA Model Selection: AIC-driven selection across competing AR, MA, and ARMA models, identifying ARMA(2,1) on the differenced log series as the superior fit.Forecasting: 12-step-ahead forecasting with 95% prediction intervals, successfully capturing the repeating $\sim$10-year oscillation.Key FindingsNo Linear Trend: Both non-parametric smoothing and formal GLS regression confirm there is no statistically significant time trend in the population dynamics over the 114-year period.ARMA(2,1) Superiority: The ARMA(2,1) model decisively outperforms competing specifications ($\Delta$AIC $\approx$ 39 over the next best model). Its complex autoregressive roots perfectly mathematically capture the $\sim$9.6-year ecological cycle.Valid Inference Requires GLS: Naive OLS regressions on this data produce invalid standard errors due to highly correlated residuals ($\hat{\rho} \approx 0.82$). GLS is required for valid inference.Requirements and ExecutionPrerequisitesYou will need R installed on your machine along with the following packages:nlme (for Generalized Least Squares estimation)tseries (for Augmented Dickey-Fuller unit-root testing)To compile the manuscript, you will need a LaTeX distribution (e.g., TeX Live, MiKTeX, or an online editor like Overleaf).How to RunClone the repository to your local machine.Set your working directory in R to the cloned folder.Run the lynx_analysis.R script.Note: The script will automatically install missing packages and save all required .pdf figures directly to your working directory.Compile paper.tex using your preferred LaTeX compiler (e.g., pdflatex or xelatex). The compiler will automatically pull in the PDFs generated by the R script.
